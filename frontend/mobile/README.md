Expo / React Native mobile application for the Couture platform.

## Local development

Once the Expo project is initialized, start the dev server with:

```bash
cd frontend/mobile
npx expo start          # scan QR code with Expo Go on your phone
npx expo start --web    # open in browser (via react-native-web)
npx expo start --ios    # open in iOS Simulator
npx expo start --android # open in Android Emulator
```
